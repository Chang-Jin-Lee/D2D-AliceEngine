# AliceEngine
D2D 기반으로 개발한 2D 게임 엔진입니다.  
C++, 2D 캐릭터 애니메이션, 게임 메커니즘(게임오브젝트, 충돌처리, 리소스매니저, 컴포넌트, 씬)을 지원합니다.
Visual Studio 2022 버전이 필요합니다 

---

## 프로젝트 구조
- `Engine/` : 엔진 핵심 모듈
- `Game/` : 게임 콘텐츠 빌드
